# jenkins-docker-pipe

## Título
CI/CD con Jenkins + Docker + GitHub

## Descripción
He desarrollado un pipeline combinando Jenkins + Docker + GitHub donde Jenkins automatiza la construcción, dockerización y despliegue de una aplicación web, simulando un entorno real de integración continua.

## Herramientas utilizadas 
- GitHub para el repositorio donde se almacena la estructura de carpetas y los archivos.
- Jenkins
- Docker Desktop / Dockerhub
- CMD o consola de PowerShell para la ejecución de comandos.
- Notepad para la creación de los archivos Dockerfile y Jenkinsfile.
